Python OOP stands for Object-Oriented Programming.

A class is a blueprint for creating objects. An object is an instance of a class.

Inheritance allows one class to reuse properties and methods from another class. The class being inherited from is called the parent class. The class that inherits is called the child class.

Polymorphism means the same method name can behave differently depending on the object.

Encapsulation means hiding internal data and controlling access using methods.

Abstraction means hiding complex implementation details and showing only the important features.
